# To do List

## Descrição
Aplicação simples para organizar as tarefas do dia a dia
O aplicativo permite que o usuário adicione a tarefa, marque como concluida e exclua.

## Tecnologias utilizadas
- HTML
- CSS
- JS

## Funcionalidades
- Adicionar tarefa
- Deletar tarefa
- Marcar como concluida
- Salva no localStorage
- Inteface simples

## Como executar o projeto
1ª - Instale um IDE, de preferência o VSCode porque roda muito bem com HTML,CSS e JS.

2ª - Copie o código disponibilizado

3ª - Execute a aplicação!
